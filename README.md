# Path of Exile Macro Collection

## Description
This project contains a collection of AutoHotkey (AHK) scripts designed to automate various tasks and improve quality of life in the game Path of Exile.

## Installation and Usage
1.  Ensure you have AutoHotkey v2.0 installed.
2.  Download all `.ahk` files into the same directory.
3.  Run `main.ahk`. The script will run in the background.
4.  Ensure Path of Exile is the active window for most macros to function correctly.

## Hotkeys

| Hotkey        | Function                                  | Description                                                              |
| :------------ | :---------------------------------------- | :----------------------------------------------------------------------- |
| `F1`          | `Hideout()`                               | Activates the Hideout function (details in `hideout.ahk`).               |
| `F2`          | `Logout()`                                | Logs out of the game (details in `logout.ahk`).                          |
| `F8`          | `Oos()`                                   | Activates the Out-of-Sync recovery function (details in `oos.ahk`).      |
| `F12`         | `Afk()`                                   | Toggles AFK status (details in `afk.ahk`).                               |
| `F2`          | `StuckKeys("NumpadHome")`                 | Resolves stuck keys, specifically for "NumpadHome" (details in `stuck_keys.ahk`). |
| `1`           | `MultiKeysRand("1", "2", "3", "4", "5")`  | Randomly presses keys "1" through "5" (details in `multi_keys_rand.ahk`). |
| `Ctrl + Shift + F` | `SetAutoflask()`                     | Sets up auto-flask functionality (details in `auto_flask.ahk`).          |
| `Ctrl + F`    | `ToggleAutoFlask()`                       | Toggles the auto-flask feature on/off (details in `auto_flask.ahk`).     |
| `Ctrl + G`    | `ToggleLoopKeys()`                        | Toggles looping key presses on/off (details in `loop_keys.ahk`).         |
